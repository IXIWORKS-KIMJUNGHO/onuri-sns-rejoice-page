# SNS Rejoice 가이드 페이지 PRD (Product Requirements Document)

## 1. 개요

### 1.1 프로젝트명
**SNS Rejoice Guide** - 온누리교회 SNS Rejoice 순장 가이드 페이지

### 1.2 프로젝트 목적
SNS Rejoice 순장님들이 순모임을 효과적으로 운영할 수 있도록 필요한 모든 정보와 자료를 한 곳에서 제공하는 올인원 가이드 허브

### 1.3 타겟 사용자
- SNS Rejoice 순장님들
- 다락방장님들
- SNS Rejoice 리더십

### 1.4 참고 사이트
- https://onnuri-seed.github.io/seed-info/ (SNS CROSS SEED 13기 가이드)

## 2. 반응형 레이아웃 구성

### 2.1 개요
하나의 웹페이지에서 접속 디바이스에 따라 **Web 레이아웃**과 **Mobile 레이아웃**을 각각 다르게 제공

| 뷰포트 | 대상 디바이스 | 레이아웃 |
|--------|---------------|----------|
| **768px 이상** | 데스크탑, 태블릿 가로 | Web 레이아웃 |
| **767px 이하** | 모바일, 태블릿 세로 | Mobile 레이아웃 |

### 2.2 Web vs Mobile 레이아웃 차이점

```
┌─────────────────────────────────────────────────────────────────┐
│             같은 웹페이지, 다른 레이아웃                          │
├─────────────────────────────┬───────────────────────────────────┤
│   Web (768px 이상)          │   Mobile (767px 이하)             │
├─────────────────────────────┼───────────────────────────────────┤
│ • 가로 넓은 화면 활용        │ • 세로 스크롤 중심                │
│ • 상단 네비게이션 바         │ • 햄버거 메뉴 (또는 하단 탭)      │
│ • 그리드 레이아웃 (2~4열)    │ • 단일/2열 레이아웃              │
│ • 호버 효과 활용             │ • 터치/스와이프 인터랙션          │
│ • 사이드바 가능              │ • 풀스크린 카드 형태              │
└─────────────────────────────┴───────────────────────────────────┘
```

### 2.3 기능 목록

**Phase 1: 가이드 페이지**
- About Rejoice
- 순모임 가이드
- 매주 해야하는 일
- 순모임 장소
- 행사 정보
- 순모임 콘텐츠

**Phase 2: EveryThanks (추후)**
- 로그인/회원가입
- 대시보드
- Always! Every Thanks (매일 감사)
- EveryThanks! Rejoice (리조이스 여정)
- 사진첩 갤러리

## 5. Web 섹션별 상세 요구사항

### 5.1 Header
- **로고**: SNS Rejoice 로고/텍스트
- **네비게이션**: 각 섹션으로 스크롤 이동하는 메뉴 버튼
- **스타일**: 스크롤 시 상단 고정 (sticky header)

### 5.2 Hero Section
- **제목**: "SNS Rejoice" 또는 "SNS Rejoice 13기"
- **부제목**: 리조이스 슬로건 또는 환영 메시지
- **배경**: 대표 이미지 또는 그라데이션

### 5.3 About Rejoice
- **목적**: Rejoice 그룹 소개
- **내용**:
  - Rejoice란?
  - 비전 및 목표
  - 조직 구조 (다락방 → 순)
  - 리더십 소개 (선택)

### 5.4 순모임 가이드
- **목적**: 순모임 운영 방법 안내
- **내용**:
  - 순모임 진행 순서
  - 첫 만남 가이드
  - 순모임 준비 체크리스트
  - 순장의 역할
  - 팁 & 노하우

### 5.5 매주 해야하는 일
- **목적**: 순장의 주간 업무 안내
- **내용**:
  - 주간 일정표/타임라인
  - 체크리스트 형태로 제공
  - 예시:
    - [ ] 순모임 일정 공지
    - [ ] 큐티 나눔 독려
    - [ ] 출석 체크
    - [ ] 기도제목 수합
    - [ ] 다락방장 보고

### 5.6 순모임 장소
- **목적**: 모임 가능한 장소 정보 제공
- **내용**:
  - 장소 목록 (카드 형태)
  - 각 장소별 정보:
    - 장소명
    - 주소
    - 수용 인원
    - 특징/분위기
    - 지도 링크 (네이버/카카오맵)
    - 사진 (선택)

### 5.7 행사 정보
- **목적**: 주요 행사 일정 안내
- **내용**:
  - 행사 캘린더 또는 타임라인
  - 각 행사별 정보:
    - 행사명
    - 일시
    - 장소
    - 대상
    - 준비물/참고사항

### 5.8 순모임 콘텐츠
- **목적**: 순모임에 활용할 수 있는 자료 모음
- **내용**:
  - 카테고리별 정리:
    - 아이스브레이킹 게임
    - 찬양 목록/링크
    - 말씀 자료
    - 기도 가이드
    - 나눔 질문
  - 외부 링크 연결 (유튜브, 드라이브 등)

### 5.9 Footer
- **내용**:
  - 온누리교회 정보
  - SNS Rejoice 연락처/문의
  - Copyright

## 6. Phase 2 상세 요구사항 (EveryThanks)

### 6.1 로그인/회원가입
- **로그인**
  - 아이디: 전화번호
  - 비밀번호: 입력
  - 슬로건: "하나님을 사랑하는 리조이스의 매일감사"

- **회원가입**
  - 아이디: 전화번호
  - 비밀번호: 6자 이상 영문/숫자 포함
  - 이름
  - 닉네임
  - 다락방: 드롭다운 (다락방장 이름 목록)
  - 순: 드롭다운 (선택한 다락방에 속한 순장 목록)
  - 완료 시 알림: "관리자 승인 후에 가입이 완료됩니다."

### 6.2 대시보드
- **순 선택 드롭다운**
  - 어드민: 모든 그룹장~순까지 표시
  - 다락방장: 본인 + 소속 순장들
  - 순장: 본인만

- **메뉴 버튼**
  - Always! Every Thanks - 매일 감사하는 우리순
  - EveryThanks! Rejoice - 13기 리조이스의 여정
  - Always! Rejoice - 리조이스 사진첩
  - Always! OOO다락방 - 우리 다락방 사진첩
  - Always! OOO순 - 우리순 사진첩

### 6.3 Always! Every Thanks (매일 감사)
- **레퍼런스**: 구글 포토 스타일
- **기능**:
  - 날짜별 그룹핑 (오늘부터 시작)
  - 각 항목: 닉네임 + 내용
  - 무한 스크롤 (5일씩 로드)
  - 필터: 날짜/닉네임
  - 수정/삭제 가능

### 6.4 EveryThanks! Rejoice (리조이스 여정)
- **UI**: 캐러셀 (좌우 화살표)
- **스타일**: 픽셀 느낌의 사진
- **내용**: 사진 + 사진설명

### 6.5 사진첩 갤러리 (Rejoice/다락방/순)
- **레퍼런스**: 구글 포토 갤러리
- **레이아웃 옵션**:
  - 1안: 세로 스크롤 (큰 사진)
  - 2안: 그리드 (3열)
- **필터**: 날짜/제목
- **페이지네이션**: 페이지 방식

### 6.6 권한 체계
| 역할 | 볼 수 있는 범위 |
|------|-----------------|
| 어드민 | 전체 (모든 다락방, 모든 순) |
| 다락방장 | 본인 다락방 + 소속 순들 |
| 순장 | 본인 순만 |
